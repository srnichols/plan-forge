LINKEDIN INLINE DIAGRAM SOURCES
================================

These Mermaid diagrams are referenced by [IMAGE] markers in the LinkedIn article .txt files.
To generate PNG images for LinkedIn upload:

Option 1: Use https://mermaid.live — paste the markup, export as PNG (1200px wide)
Option 2: Use VS Code Mermaid Preview extension
Option 3: Use mmdc CLI: npx @mermaid-js/mermaid-cli -i file.mmd -o file.png -w 1200

Recommended: Export at 1200px wide, light background, for LinkedIn readability.
