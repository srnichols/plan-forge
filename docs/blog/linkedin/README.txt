LINKEDIN ARTICLE VERSIONS
=========================

Instructions for posting to LinkedIn:
1. Go to https://www.linkedin.com/article/new/
2. Upload the hero image (listed per post)
3. Paste the title into the title field
4. Copy the body text and paste into the editor
5. LinkedIn preserves bold, italic, links, lists, and blockquotes from rich text
6. Add 3-5 hashtags at the bottom (suggestions included per post)

Hero images are in: docs/blog/assets/
Each .txt file below is one article, ready to paste.
