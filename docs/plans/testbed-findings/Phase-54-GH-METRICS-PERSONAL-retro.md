## Friction Log

1. **Endpoint interface mismatch**: A prior session had already written an action-based
   `/api/github-personal` endpoint (`?action=profile|repo|coauthors`) that did not match
   the plan-specified unified response shape (`{ ok, user, repo, copilotSignal, errors, _meta }`).
   S1 replaced the endpoint entirely and rewrote the test file. Time cost: ~15 min.

2. **Test file pre-existing with wrong assertions**: `github-personal-rest.test.mjs` existed
   from the prior session and asserted against the old interface (403/404/429 responses).
   Required a full rewrite. Tests now assert 200-only responses with structured errors.

3. **No friction**: The module-level exports, ESM mock-gh harness, and window-attach pattern
   all worked first-try. The clean separation between `github-personal.mjs` and
   `github-metrics.mjs` prevented any merge conflict.
