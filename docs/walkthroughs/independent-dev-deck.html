MCP Tools — AI Calls Plan Forge

Claude, Cursor, and Copilot call forge operations as native functions.

  1. 1 Install the MCP server:
    cd mcp && npm install
  2. 2 In Claude Code, type: "Run forge_smith" → diagnostics appear
  3. 3 Then: "Run forge_sweep" → zero markers confirmed

Your AI agent discovers these tools automatically. No terminal commands needed — the agent calls forge operations the same way it calls any other function.

106 MCP tools available — diagnostics, plan analysis, cost reporting, tempering, memory, and more.
