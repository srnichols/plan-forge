            📚 About the Inner Loop
          
          

            
The Inner Loop is a set of feedback subsystems that let Plan Forge improve without human intervention between slices.

            
    
              
  • Reviewer calibration — shows when local reviews become eligible to adjust grader confidence.
    • 
              
  • Gate suggestions — counts of accepted validation-gate commands, ready for promotion.
    • 
              
  • Cost anomalies — slices whose token spend drifted above the per-model median.
    • 
              
  • Proposed fixes — patches produced by the auto-fix subsystem awaiting review.
    • 
              
  • Federation — recent trajectories discovered in sibling repos configured under brain.federation.
    • 
            

            
All subsystems are advisory by default. Enable or tune them in the Config tab.
